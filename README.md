# Diwali Sales Analysis using Python
Exploratory Data Analysis of Diwali sales data using Python to uncover customer purchasing patterns and business insights.

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand customer purchasing behavior during the festive season.

The analysis helps identify:
- Top customer segments
- Most profitable states
- Popular product categories
- Buying patterns based on age and gender

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains sales transactions during the Diwali festival including customer demographics, product categories, and purchase amounts.

## Data Cleaning
The following steps were performed:
- Removed unnecessary columns
- Checked missing values
- Converted data types
- Handled null values
