# Diwali Sales Analysis using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data to understand customer purchasing behavior during the festive season.

The analysis helps identify:
- Top customer segments
- Most profitable states
- Popular product categories
- Buying patterns based on age and gender

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains sales transactions during the Diwali festival including customer demographics, product categories, and purchase amounts.

## Data Cleaning
The following steps were performed:
- Removed unnecessary columns
- Checked missing values
- Converted data types
- Handled null values

## Exploratory Data Analysis

### Gender Analysis
Analyzed purchase behavior based on gender.

Key Insight:
Female customers made more purchases compared to male customers.

### Age Group Analysis
Analyzed which age group contributes most to sales.

Key Insight:
Age group **26-35** contributed the highest number of purchases.

### State Analysis
Identified states with the highest number of orders.

Key Insight:
Top states by orders:
1. Uttar Pradesh
2. Maharashtra
3. Karnataka

### Occupation Analysis
Analyzed which occupations spend more during Diwali.

Key Insight:
IT, Healthcare and Aviation sectors show higher spending.

### Product Category Analysis
Identified most popular product categories.

Key Insight:
Food, Clothing and Electronics categories dominate sales.

## Key Business Insights
- Female customers are the primary buyers during Diwali.
- Age group 26–35 contributes the highest revenue.
- Uttar Pradesh and Maharashtra generate the highest orders.
- Certain product categories dominate festive sales.

## Conclusion
This analysis helps businesses understand customer purchasing patterns and target marketing strategies during festive seasons.
